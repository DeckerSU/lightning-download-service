Lightning Download Service

All files are free. You are paying only a small fee for file storage and serving services.

All payments at this stage can be made only with Lightning Satoshis.

The fastest way to start using Lightning is by installing the Muun Wallet and topping it up with some Satoshis on-chain or off-chain (e.g., via BTC).

Important: After you create the invoice, do not close the page or invoice window until the payment is settled. You will receive the download link right after your payment is completed.

Download is allowed only once. After pressing the download button, you will be able to save the file once. If you miss this opportunity (e.g., close the window or something else happens), you will need to create a new invoice and pay again.
